<h1 align="center">
 GPT Terminal
</h1>

# Clone the repo
```
git clone https://github.com/xmahbub/GPT-Terminal-Support.git
```
# Go to GPT-Terminal-Support
```
cd GPT-Terminal-Support
```

# Give install file execute privilege
```
chmod +x install
```
# Install the application
```
sudo ./install
```
# Use 
```
gpt "hello ...your search"
```
# Note: jq is required

## If Ubuntu
```
apt install jq
```
## If RedHat
```
yum install jq
```
