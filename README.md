<h1 align="center">
 GPT Terminal
</h1>

# Give file execute privilege
```
chmod +x install
```
# Install the application
```
./install
```
# Use 
```
gpt "hello ...your search"
```
# Note: jq is required

## Ubuntu
```
apt install jq
```
## RedHat
```
yum install jq
```
